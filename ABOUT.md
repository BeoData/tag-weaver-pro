# About TagWeaver Pro

**Professional AI-powered MP3 metadata editor for DJs, music producers, and audio professionals.**

## Quick Description

TagWeaver Pro is a modern, web-based MP3 metadata editor that combines:
- **ID3 Tag Editing** - Clean, professional metadata management
- **AI Metadata Cleanup** - Automatically detect and remove AI-generated traces
- **BPM Detection** - Automatic tempo analysis for DJ preparation
- **Musical Key Detection** - Camelot key support for harmonic mixing
- **Waveform Visualization** - Canvas-based audio visualization
- **Batch Processing** - Process multiple files at once

Built with React 18, TypeScript, Vite, Tailwind CSS, and Supabase.

Perfect for: DJs, music producers, audio professionals, collectors, radio stations, and podcast creators who need clean, consistent music metadata.
