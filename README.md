## Overview

**TagWeaver Pro** is a modern audio metadata editor focused on fast, clean, and reliable MP3 library management.

It helps users clean messy ID3 tags, remove unwanted AI-generated metadata, detect BPM, analyze musical keys, manage cover artwork, visualize waveforms, and process large music libraries with batch tools.

TagWeaver Pro is built for practical music workflows where clean metadata matters: DJ preparation, music production, radio libraries, podcast archives, audio collections, and AI-generated music cleanup.

---

## Why TagWeaver Pro?

Managing thousands of MP3 files manually is slow and inconsistent. TagWeaver Pro brings metadata editing, AI-assisted cleanup, audio analysis, waveform visualization, and batch processing into one streamlined application.

It is designed for:

- DJs preparing sets and playlists
- Music producers managing exported tracks
- Collectors organizing large MP3 libraries
- Radio stations maintaining clean archives
- Podcast creators managing audio metadata
- Audio professionals who need consistent file information

## Features

### AI Metadata Cleaner

Automatically detect and clean unwanted metadata patterns commonly found in AI-generated or software-processed audio files.

- Remove AI-generated metadata traces
- Clean tags such as `Generated by AI`, `SUNO`, and similar markers
- Normalize artist, title, album, and genre fields
- Detect inconsistent metadata patterns
- Review changes before applying them

### BPM Detection

Analyze audio files and automatically detect tempo.

- Automatic BPM detection
- Fast audio analysis
- Useful for DJ preparation and library sorting
- Supports single-file and batch workflows

### Musical Key Detection

Detect musical key information for harmonic mixing and better track organization.

- Musical key detection
- Major and minor key support
- Camelot key support
- Store detected key data in metadata

### Professional ID3 Tag Editing

Edit common MP3 metadata fields through a clean interface.

- Title
- Artist
- Album
- Genre
- Year
- Track number
- Composer
- Comments
- Cover artwork
- Custom metadata fields

Supports ID3v2.3 and ID3v2.4 workflows.

### Waveform Visualization

View audio files with a modern waveform display.

- Canvas-based waveform rendering
- Smooth visual feedback
- Scanner-style animation
- Optimized for modern browsers

### Batch Processing

Process multiple MP3 files at once.

- Bulk metadata cleanup
- Bulk tag editing
- Batch BPM detection
- Batch musical key detection
- Batch cover artwork updates
- Consistent formatting across folders

### Reports and Export

Generate reports for processed files.

- CSV export
- TXT export
- Processing logs
- Change summaries

## Installation

### Requirements

- Node.js 18 or newer
- npm, yarn, or bun
- Supabase project, if authentication or cloud features are enabled

### Clone the Repository

```bash
git clone https://github.com/BeoData/tag-weaver-pro.git
cd tag-weaver-pro
```

### Install Dependencies

Using npm:

```bash
npm install
```

Using yarn:

```bash
yarn install
```

Using bun:

```bash
bun install
```

### Configure Environment Variables

Create a local environment file:

```bash
cp .env.example .env
```

Update `.env` with your configuration:

```env
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
VITE_API_URL=http://localhost:3000
VITE_APP_ENV=development
```

### Start Development Server

Using npm:

```bash
npm run dev
```

Using yarn:

```bash
yarn dev
```

Using bun:

```bash
bun dev
```

Open the application:

```text
http://localhost:5173
```

## Usage

### Edit MP3 Metadata

1. Upload an MP3 file.
2. Review existing metadata.
3. Edit title, artist, album, genre, year, artwork, and other fields.
4. Save the updated file.

### Clean Metadata

1. Select one or more MP3 files.
2. Run the metadata cleaner.
3. Review detected issues.
4. Apply or reject suggested changes.

### Detect BPM and Key

1. Upload an audio file.
2. Start analysis.
3. Review detected BPM and musical key.
4. Save results to file metadata.

### Process Files in Bulk

1. Select multiple files or a folder.
2. Choose the desired operation.
3. Run batch processing.
4. Export a report if needed.

## FAQ

### Does TagWeaver Pro change audio quality?

No. TagWeaver Pro focuses on metadata editing and analysis. It does not re-encode or degrade audio quality.

### Does it support batch processing?

Yes. TagWeaver Pro supports bulk workflows for metadata cleanup, editing, BPM detection, and key detection.

### Can it detect Camelot keys?

Yes. Musical key detection includes Camelot key support for DJ workflows.

### Can I edit cover artwork?

Yes. You can add, replace, or manage album artwork.

### Does it work offline?

Most metadata editing and local analysis features can work locally. Cloud-dependent features require the configured backend services.
