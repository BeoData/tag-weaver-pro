**TagWeaver Pro** is a professional-grade MP3 metadata editor designed for DJs, music producers, and audiophiles. It features AI-powered metadata cleaning, automatic BPM/Key detection, and a sleek, modern waveform visualizer.

## ğŸš€ Key Features

-   **AI Metadata Cleaner**: Automatically identifies and removes traces of AI generation and software-specific tags (like "Generated by AI", "Lovable", etc.).
-   **Auto BPM & Key Detection**: Integrated audio analysis using FFT to detect BPM and Musical/Camelot keys.
-   **Dynamic Waveform Visualizer**: High-performance canvas-based waveform with a modern "scanner" sweep animation.
-   **Professional ID3 Editing**: Full support for ID3v2.3 and ID3v2.4 tags, including cover art management.
-   **Tiered Access System**: Built-in authentication (Supabase) with Free, Basic, and Pro tiers.
-   **Bulk Processing**: Process entire folders of music in seconds (available for Basic and Pro users).
-   **Exportable Reports**: Generate CSV or TXT reports of all metadata changes and cleanups.

## ğŸ“¦ Getting Started

### Local Development

1.  **Clone the repository**:
    ```sh
    git clone https://github.com/BeoData/tag-weaver-pro.git
    cd tag-weaver-pro
    ```

2.  **Install dependencies**:
    ```sh
    npm install
    ```

3.  **Setup Environment Variables**:
    Create a `.env` file based on `.env.example` (or the deployment guide) with your Supabase credentials:
    ```env
    VITE_SUPABASE_URL=your_project_url
    VITE_SUPABASE_ANON_KEY=your_anon_key
    ```

4.  **Run the dev server**:
    ```sh
    npm run dev
    ```
